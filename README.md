# ЛР5: CRUD пользователей, авторизация и отчёты

## Запуск

```powershell
pip install -r requirements.txt
flask --app app run --debug
```

Если команда `python` в Windows указывает на Microsoft Store, используйте установленный интерпретатор Python напрямую.

## Начальная учётная запись

При первом запуске приложение автоматически создаёт роли и администратора:

- логин: `admin`
- пароль: `Admin123!`

База данных SQLite создаётся автоматически в каталоге `instance`.
